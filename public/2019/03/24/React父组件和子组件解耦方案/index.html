背景

解耦方案

一、使用 render props

可以看到，通过 render props 的设计实现了父组件和子组件之间的解耦。

二、Compound Component

可以看到 compound component 也能实现父子组件解耦，只是在这里可能不是那么完美。其实 compound component 的应用远不止如此，读者可以自行挖掘。😃
